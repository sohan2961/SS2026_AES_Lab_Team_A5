# 04_Arduino_Relay_Node_DeepakKapil

Owner: Deepak Kapil

Purpose: Arduino Uno WiFi Rev2 receives relay commands and controls the KY-019 relay. The relay is currently used to simulate an actuator using an LED. In a future version, it can control a real water pump, fan, or light.

Tasks:
- Connect KY-019 relay module.
- Receive MQTT commands.
- Turn relay ON/OFF.
- Publish relay status.
