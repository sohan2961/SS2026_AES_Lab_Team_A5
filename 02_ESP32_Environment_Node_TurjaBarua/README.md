# 02_ESP32_Environment_Node_TurjaBarua

Owner: Turja Barua

Purpose: ESP32-1 reads KY-015 temperature/humidity sensor, shows values on I2C LCD, reads rotation sensor as threshold input, and sends data using MQTT.

Tasks:
- Read KY-015 sensor.
- Display values on I2C LCD.
- Read rotation sensor.
- Publish MQTT data to Raspberry Pi.
